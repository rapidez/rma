# Rapidez RMA

MageOS_RMA compatibility

## Installation

```
composer require rapidez/rma
```

## Configuration

You can publish the config with:
```
php artisan vendor:publish --tag=rapidez-rma-config
```

## Views

You can publish the views with:
```
php artisan vendor:publish --tag=rapidez-rma-views
```
